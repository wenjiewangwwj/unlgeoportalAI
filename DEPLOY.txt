CAN GITHUB HOST THE PYTHON API?
  No. GitHub Pages only serves static files (HTML/CSS/JS). It cannot run
  uvicorn/FastAPI. Your API must run on a "Python web host" (Render, Railway,
  Azure, a VM, etc.). You still use GitHub to STORE the code.

CAN THE HTML LIVE ON GITHUB AND WORK?
  Yes, if the HTML calls your API by full URL (API_BASE in index.html or ?api=).
  You must also set CORS_ORIGINS on the API to include your GitHub Pages origin,
  e.g. https://yourname.github.io

OPTION A — RENDER.COM (FREE TIER, HTTPS)
  1. Push this repo to GitHub.
  2. Sign up at https://render.com and connect GitHub.
  3. New → Web Service → pick the repo.
  4. Render can read render.yaml; or set manually:
       Build:  pip install -r requirements.txt
       Start:  uvicorn backend.main:app --host 0.0.0.0 --port $PORT
  5. In Render → Environment, add (examples):
       PORTAL_SHARING_REST=https://geoportal.nead.nebraska.edu/portal/sharing/rest
       LLM_PROVIDER=gemini
       GEMINI_API_KEY=(from https://aistudio.google.com/apikey )
       GEMINI_MODEL=gemini-2.0-flash
       CORS_ORIGINS=https://YOURNAME.github.io,http://127.0.0.1:5500
     Copy any other variables from env.example as needed.
  6. Deploy. Copy the service URL, e.g. https://geoportal-nl-search.onrender.com

GITHUB PAGES FOR THE UI
  1. Repo → Settings → Pages → Build from branch.
  2. Use "main" + "/docs" OR "/ (root)" — GitHub only allows those folders for
     Pages from a branch. This project keeps the page in frontend/index.html.
     Easiest: copy frontend/index.html to docs/index.html (create docs/), commit,
     set Pages source to /docs on main.
  3. In docs/index.html (or wherever you publish), set:
       const API_BASE = "https://YOUR-SERVICE.onrender.com";
  4. Wait for Pages URL (https://YOURNAME.github.io/REPO/) and add that exact
     origin to CORS_ORIGINS on Render, then redeploy or restart the service.

NOTES
  - Free Render apps may "spin down" after idle; first request can be slow.
  - Never commit .env or API keys; set secrets only in Render / host dashboard.
